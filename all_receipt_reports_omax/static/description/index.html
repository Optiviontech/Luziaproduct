Receipt Reports For Sales, Accounting, Purchase, Inventory,Payment

Overview

This module helps user to print Odoo standard reports in receipt format. This module help to print sales,accounting,purchase,inventory and payment standard report in receipt(slip) format.

Feature

  • Print Receipt(slip) for Quotations and Sales Order.
  • Print Receipt(slip) for RFQ and Purchase Order.
  • Print Receipt(slip) for Customer Invoice and Vendor Bills.
  • Print Receipt(slip) for Customer Payment and Vendor Payment.
  • Print Receipt(slip) for Picking Operations and Delivery Slip.
  • Print Receipt(slip) for Customer Credit Note and Customer Sales Receipt.
  • Print Receipt(slip) for Vendor Refund and Purchase Receipt.
